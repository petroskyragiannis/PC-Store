## Επιχειρησιακοί Κανόνες

| Επιχειρησιακοι Κανονες | Περιγραφη |
| ------ | ------ |
| ΕΚ1 | Το username ειναι μοναδικο για καθε πελατη |
| ΕΚ2 | Η ηλικια του πελατη πρεπει να ειναι μεγαλυτερη των 18 |
| ΕΚ3 | Ο κωδικος πρεπει να εχει τουλαχιστον 8 ψηφια |
| ΕΚ4 | Ολα τα στοιχεια του πελατη ειναι υποχρεωτικα |
